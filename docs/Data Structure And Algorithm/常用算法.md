>## 欧几里得算法（辗转相除法）
>求最大公因数和最小公倍数
```cpp
//最大公因数
int GCD(int a, int b){
	if(b == 0) return a;
	else return GCD(b, a%b);
}
//最小公倍数
int LCM(int a,int b){
    int gcd = GCD(a,b);
    int lcm = a/gcd*b;
    return lcm;
}
```
